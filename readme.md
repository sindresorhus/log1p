# log1p

> ES2015 [`Math.log1p()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/log1p) [ponyfill](https://ponyfill.com)


## Install

```
$ npm install --save log1p
```


## Usage

```js
var log1p = require('log1p');

log1p(5);
//=> 1.791759469228055
```
